# 👋 Start here

**You're in the right place.** This is your AI second brain — an Obsidian vault that Claude lives inside.

## To set it up (just once)

1. Make sure the **Claudian** plugin is installed and you're logged in to Claude — if not, see **`README.md`**.
2. Open the **Claudian panel** (the Claude icon on the right sidebar).
3. Type exactly:

   > **`set this up`**

Claude will ask you a few quick questions (~5 minutes) and build the rest *for* you.

> **Nothing happened?** Type: **`read CLAUDE.md and set this up`**
